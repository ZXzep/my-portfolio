# Portfolio Website - Punnathat Samoprong

A dark futuristic portfolio website built with HTML, CSS, and vanilla JavaScript.

## Features

- **5 Pages**: Home, About, Portfolio, Project, Contact
- **Dark Futuristic Design**: Modern UI with neon glow effects
- **Particle Background**: Animated floating particles on all pages
- **Responsive Design**: Works on all devices
- **Smooth Animations**: CSS transitions and JavaScript interactions
- **Portfolio Filter**: Filter projects by category (UX/UI, Graphic, 3D)
- **Contact Form**: Ready-to-use contact form

## Project Structure

## Design System

- **Base Color**: #0A0A0A
- **Accent Color**: #7B45FF (blue-purple neon)
- **Text Colors**: #FFFFFF, #A8B0C0, #6F7480
- **Fonts**: Space Grotesk (headings), Inter (body)

## Deployment

This project is ready to deploy to Netlify:

1. Push your code to GitHub
2. Connect your repository to Netlify
3. Deploy!

The site will work immediately with no build process required.

## Customization

### Update Contact Information
Edit `contact.html` and replace:
- Email address
- LINE ID
- Social media links

### Add Your Projects
1. Edit `portfolio.html` to add portfolio items
2. Update `project.html` with your project data
3. Replace placeholder images with your actual project images

### Update Resume
Add your resume file and update the download link in `about.html`
